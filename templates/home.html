Automated Tool to Test ATA HDD Connected with RAID Card

This is an automated tool which monitors hard drives for health status, taking advantage of S.M.A.R.T. This tool is capable of check system hard drive connected with ATA controller and hard drives connected with RAID controller. The application checks S.M.A.R.T attributes and S.M.A.R.T Error log to ensure hard drive is healthy. This tool also perform Interface test, Seek test, Read/Write test and error verification. It is also able to enable and disable certain settings of your hard drive, settings which are normally only available to advanced users.
